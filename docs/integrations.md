# Integrations

`operator` can be used through multiple technical surfaces.

## Current public entrypoints

- CLI commands
- JSON-capable CLI output surfaces
- Python package imports for selected stable modules
- ACP-related client/runtime utilities

## Start points

- CLI usage: [reference/cli.md](reference/cli.md)
- Python API reference: [reference/python/overview.md](reference/python/overview.md)

## Design-level integration direction

For the deeper design vision around agent integration, see
`design/AGENT-INTEGRATION-VISION.md` in the repository.

That document is design authority and future-facing integration rationale, not the primary
quickstart surface.
