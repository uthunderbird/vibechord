# Design Corpus

This directory contains the committed design authority and design-history corpus for `operator`.

It is intended for contributors and deep readers who want product rationale, architecture,
decision provenance, and design-process artifacts.

## What lives here

- `VISION.md` — product and behavioral design authority
- `ARCHITECTURE.md` — structural implementation authority
- `*-UX-VISION.md` — design specs for user surfaces
- `adr/` — architectural decisions
- `rfc/` — broader design specifications
- `brainstorm/` — design exploration
- `internal/` — critiques and internal design notes
- implementation plans, critiques, and roadmaps

## What does not live here

This directory is not the primary end-user docs surface.

For public product docs, start with:

- [`../README.md`](../README.md)
- [`../docs/`](../docs/)

1. [`VISION.md`](VISION.md)
